## Installation

1. Clone the Treads repo
```sh
git clone https://github.com/sparrill14/Treads.git
```
2. Install node packages
```sh
npm install
```

## Usage
1. Run the server in development mode
```sh
npm run start
```
2. Build the project in production mode
```sh
npm run build
```
